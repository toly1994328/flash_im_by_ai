## 变更摘要

新增好友系统：搜索用户、发送好友申请（附留言）、接受/拒绝申请、好友列表、删除好友、扫码添加。好友申请和接受通过 WebSocket 实时推送通知，接受好友后自动创建私聊会话并发送打招呼消息。

后端新建 im-friend 独立 crate（10 个接口 + 3 种 WS 帧类型），前端新建 flash_im_friend 模块（微信风格通讯录 + 好友详情 + 添加朋友 + 搜索 + 扫码 + 申请管理）。

附带变更：所有脚本从 PowerShell 迁移为 Python（跨平台），新增 feature-sync steering。
