# 认证与用户域 — 局域网络

涉及节点：I-01~I-04, F-01~F-03

---

## 节点详情

| 编号 | 功能节点 | 模块 | 端 | 职责 |
|------|---------|------|-----|------|
| I-01 | 应用状态管理 | flash-core | 后端 | AppState，数据库连接池，被所有后端模块依赖 |
| I-02 | 手机号登录 | flash-auth | 后端 | 手机号 + 验证码/密码登录 |
| I-03 | Token 签发与验证 | flash-auth | 后端 | JWT 签发、中间件验证 |
| I-04 | 用户资料管理 | flash-user | 后端 | 用户资料 CRUD |
| F-01 | 登录注册页 | flash_auth | 前端 | 手机号输入、验证码、密码登录 |
| F-02 | 用户会话状态 | flash_session | 前端 | token 管理、用户信息缓存 |
| F-03 | 启动流程 | flash_starter | 前端 | 恢复会话 → 路由决策 |

---

## 边界接口

### HTTP 接口

| 接口 | 提供节点 | 消费节点 |
|------|---------|---------|
| POST /auth/login | I-02 | F-01 |
| GET /users/:id/profile | I-04 | F-02 |
